# Cálculo de Média Ponderada dos Alunos

## Descrição

Programa desenvolvido para calcular a média final de três alunos utilizando média ponderada entre os graus **GA** e **GB**.

A média é calculada pela fórmula:

> Média Final = (GA × 33,33%) + (GB × 66,67%)

## Objetivo

- Armazenar as notas de 3 alunos.
- Calcular a média ponderada de cada aluno.
- Exibir o nome e a média final formatada com duas casas decimais.

## Resultado Esperado

```text
Ana: 8,67
Bruno: 6,83
Carla: 6,00
```
