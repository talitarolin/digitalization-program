# Digitalization Program – Programming Logic in Java

## 🎯 Objetivo

Este repositório reúne os exercícios e desafios desenvolvidos durante o módulo de **Lógica de Programação** do Digitalization Program.

O objetivo é consolidar os fundamentos da programação, entendendo que programar consiste em estruturar soluções para problemas por meio de algoritmos claros, organizados e reutilizáveis.

---

## 📚 Conteúdo

Os exercícios estão organizados por tópicos fundamentais:

* Variáveis e Tipos de Dados (`String`, `int`, `double`, `boolean`)
* Operadores Aritméticos
* Estruturas Condicionais (`if`, `else` e `switch`)
* Laços de Repetição (`for`, `while` e `foreach`)
* Métodos e Funções
* Debugging
* Desafio Final: Caixa Registradora

---

## 🛠 Tecnologias

* Java
* JDK 17+ (ou versão compatível)

---

## ▶️ Como executar

Compile o arquivo desejado:

```bash
javac Main.java
```

Execute o programa:

```bash
java Main
```

> Certifique-se de executar os comandos na pasta onde o arquivo está localizado.

---

## 💡 Debug

Durante os exercícios foi utilizado `System.out.println()` para acompanhar o fluxo de execução, validar variáveis e facilitar a identificação de erros.

---

## 📈 Objetivo de aprendizagem

Este repositório representa minha evolução nos conceitos fundamentais de programação em Java e serve como base para estudos mais avançados em orientação a objetos, estruturas de dados e arquitetura de software.
